## MoodleNotifTgBot

### Requirements
 - Python 3.9+
 - Poetry

### Installation
```shell
$ git clone https://github.com/RuslanUC/MoodleNotifTgBot
$ cd MoodleNotifTgBot
$ poetry install
```

Next, you need to fill out config.py or set environment variables (BASE_URL, API_ID, API_HASH, BOT_TOKEN). 

### Usage
```shell
$ poetry run python main.py 
```
